Detailed Description

Manages a collection of core modules within a ship system. This class is responsible for registering core modules, applying various operations (start-up, shut-down, reboot) to them, and retrieving information about their state and health.

◆ AddSingleModule()

void CoreShipModuleManager.AddSingleModule ( BC_CoreModule module)

Adds a core module to the internal registry.

Parameters
moduleThe core module to be added.

This method attempts to add the specified core module to the internal registry of modules managed by this class. If the module is already registered, no action is taken.

After successful registration, the method attempts to establish a link between the module and this manager. This may involve the module registering for events or accessing functionalities provided by this manager.
