◆ ReputationValue

enum FactionReputation.ReputationValue

Represents the reputation level a player or faction may have with another faction.

Enumerator
Hostile 

The faction is considered an enemy and will be attacked on sight.

Disfavored 

The faction is viewed with suspicion and may be treated with hostility.

Neutral 

The faction has no standing reputation with the other faction. They will be treated with indifference.

Cooperative 

The faction has established a positive working relationship with the other faction.

Trusted 

The faction has earned the trust and respect of the other faction.

void FactionReputation.AddFactionToReputationData ( SO_FactionData faction)

Adds a new faction to the reputation data if it doesn't already exist.

Parameters
factionThe faction data to add.

This method checks if the reputation data already contains information for the provided faction. If not, a new ReputationDatum object is created and added to the internal ReputationData list.

void FactionReputation.ModifyStanding ( SO_FactionData faction,
int value )

Modifies the reputation standing of a faction.

Parameters
factionThe faction whose reputation to modify.
valueThe amount to change the reputation by (positive for increase, negative for decrease).

This method attempts to find the reputation data for the specified faction. If found, the value is added to the existing standing. If the faction data is not found, a log message is created using the provided CustomLogger class.
